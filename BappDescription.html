A comprehensive Burp Suite extension, AI GraphQL Pentester leverages Burps native Montoya AI to generate sophisticated malicious GraphQL test queries for authorized security testing. This extension features dynamic AI status monitoring, schema introspection, RAG integration, and intelligent query formatting.

Features

Core AI-Powered Functionality:
Montoya AI Integration: The extension uses Burp Suites built-in AI capabilities (Professional edition required). Dynamic AI Status: Real-time AI status monitoring is available with automatic updates. Schema Extraction: Two methods supported, automatic introspection of GraphQL endpoints or manual schema input and parsing. AI-Powered Query Generation: There are two modes available. Bulk Generation generates multiple malicious queries targeting common vulnerabilities, and Single Query Testing generates malicious variants of a specific target query/mutation. Custom AI Messages: Direct interaction with Montoya AI for specialized security guidance.

Advanced Security Testing:
The extension tests for a comprehensive range of GraphQL vulnerabilities, including SQL injection attempts, authorization bypass vulnerabilities, DoS via deep nesting and resource exhaustion, and information disclosure through introspection. Other tests include input validation bypass, NoSQL injection attacks, time-based attack vectors, field access bypasses, rate limiting vulnerabilities, and authentication bypass in nested queries.

Advanced Features:
Single-Line Query Formatting: Automatically converts multi-line GraphQL queries to single-line format for easy testing. Dynamic Status Updates: AI status automatically refreshes when toggling the Use AI checkbox. External RAG Integration: Optional integration with RAG services for enhanced security knowledge. Smart Query Processing: Intelligent parsing of AI responses with context preservation. Unicode Handling: Robust Unicode character processing for international content. Caching System: Prevents duplicate expensive operations and RAG queries. Background Processing: Non-blocking AI operations to keep Burp responsive.

Usage

Initial Setup:
To use the extension first enable the AI Features in the Extensions → Installed section by checking the Use AI checkbox. The AI Status tab will show ENABLED when active. Then optionally, you can configure the RAG integration by setting up the external RAG endpoint.

Schema Extraction:
Automatic Introspection: Enter your GraphQL endpoint URL and click Introspect Schema. Manual Input: Paste your GraphQL schema JSON in the text area and click Parse Schema.

AI-Powered Query Generation:
Targeted Testing: Paste a specific GraphQL query/mutation in the Target Query/Mutation field, specify attack types, and click Generate Malicious Queries. Montoya AI will create malicious variants with detailed explanations. Comprehensive Testing: For bulk generation, leave the target query field empty, specify desired attack types, and click Generate Malicious Queries. Review generated test queries with attack descriptions.

Custom AI Interaction:
Direct AI Communication: Switch to the Custom Messages tab, enter specific security testing requirements, and click Send Custom Message. Receive tailored security advice and specialized queries.

Dynamic AI Status Monitoring:
Real-time Updates: AI status automatically refreshes when switching tabs. Button State Management: AI buttons enable/disable based on the current AI status. Visual Feedback: Color-coded status indicators. Instructions: Dynamic help text based on current AI availability.
